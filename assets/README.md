# Assets

Q版桌宠资源目录。

## 目录约定

- `character/`: 角色本体动画与表情
- `effects/`: 飘心、星星、烟雾、睡眠符号等特效
- `ui/`: 气泡、按钮、图标、菜单面板
- `audio/`: 点击、情绪反馈、睡眠、语音
- `config/`: 动画、状态、资源规范配置
- `temp/`: 临时导出或缓存文件

## 命名建议

- 序列帧统一使用两位数编号，如 `idle_01.png`
- 特效序列帧建议使用 `heart_01.png`、`zzz_01.png`
- 情绪贴图使用单张命名，如 `happy.png`
- 音频建议按动作或情绪拆分，例如 `click_01.wav`
